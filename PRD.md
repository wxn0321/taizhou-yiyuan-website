# 台州意园新材料科技有限公司 — 网站需求规格说明书

> 本文档是网站开发的唯一依据。任何后续添加的功能或改动，都需要先更新此文档。

---

## 一、项目概述

### 1.1 项目背景
**台州意园新材料科技有限公司**是一家位于浙江省台州市仙居县的有限责任公司，成立于2014年，注册资本580万人民币。公司主营**新材料研发与制造**，产品包括注浆材料、建筑防水嵌缝密封材料、金属波纹管、日用塑料制品等。经过10年发展，公司具备成熟的生产能力，需要建设品牌官网，用于展示企业实力、产品体系，获取工程客户询盘，拓展B2B业务渠道。

### 1.2 项目目标
1. 树立专业品牌形象，展示公司实力与产品体系
2. 展示产品目录，方便客户浏览产品规格和应用场景
3. 提供在线询盘/咨询入口，客户提交需求后由客服跟进
4. 展示工程案例和技术资料，增强客户信任
5. 支持PC端和手机端访问（响应式设计）

### 1.3 成功标准
- 网站能够正常在手机和电脑浏览器中打开
- 所有功能页面均能正常访问，无死链
- 商品展示清晰，分类明确
- 客户能够成功提交订单（模拟下单流程）
- 整体风格专业商务，有品牌质感

## 三、功能模块规划

### 3.1 首页（Home）
- **顶部导航栏**：Logo + 导航菜单（产品中心/解决方案/工程案例/关于我们/联系我们）+ 询价热线按钮
- **Hero区域**：展示品牌主视觉图 + 核心价值主张（如"专业防水·安心建筑"）+ 主打产品类别入口
- **关于我们**：简短公司介绍（公司实力、生产资质、服务范围，2-3段文字 + 配图）
- **产品分类入口**：图标 + 分类名称 + 简短描述，快速导航到对应产品分类
- **核心优势展示**：4-6个优势点（如：厂家直供 / 品质认证 / 现货供应 / 技术支持 / 物流配送 / 定制服务）
- **工程案例展示**：2-3个代表性工程案例（项目名称 + 简介 + 照片）
- **页脚**：版权信息、联系方式（电话/地址）、快速导航链接、资质证书展示

### 3.2 产品中心页面
- **分类导航**：侧边栏展示所有产品分类（根据实际经营范围调整）
  - 注浆材料类（地基加固、裂缝修补用）
  - 建筑防水嵌缝密封材料类（防水密封、接缝处理）
  - 金属波纹管类（排水、通风管道）
  - 日用塑料制品类（日常塑料配件）
- **产品列表**：网格布局展示产品卡片
  - 产品图片（主图）
  - 产品名称
  - 产品分类标签
  - 简短描述（适用场景）
  - "查看详情"按钮
- **产品详情页**：
  - 产品大图（支持点击放大/画廊切换）
  - 产品名称、分类、型号
  - 产品特点 / 优势
  - 技术参数表格（厚度、规格、材质、执行标准等）
  - 适用场景 / 应用范围
  - 施工工艺说明（如有）
  - 相关案例链接
  - "获取报价"按钮
  - "在线咨询"按钮

### 3.3 工程案例页面
- 案例列表页：卡片式布局
  - 项目封面图
  - 项目名称 + 所在地
  - 项目类型（住宅/商业/工业/基础设施）
  - 使用产品
  - 完工时间
- 案例详情页：
  - 项目概况（面积、类型、时间）
  - 项目需求背景
  - 解决方案（使用了哪些产品）
  - 施工前后对比照片
  - 客户评价（如有）

### 3.4 在线询盘（替代购物车）
- **询盘表单页面**：客户选择感兴趣的产品，填写需求信息
  - 姓名（必填）
  - 公司名称（选填）
  - 联系电话（必填，手机号格式校验）
  - 所在地区（必填，下拉选择）
  - 工程类型（选填，如：住宅/厂房/桥梁/地下室等）
  - 工程规模（选填，面积或数量）
  - 留言内容（选填，补充需求说明）
- 提交后：
  - 显示"询盘提交成功"提示
  - 记录询盘信息（本地存储 + 提示客服跟进）
  - 提供"返回首页"按钮
- ⚠️ **注意**：暂不做购物车功能，改为按需询盘模式

### 3.5 解决方案 / 应用场景
- 按行业/场景分类展示应用方案
- 场景：屋面防水 / 地下室防水 / 厨卫防水 / 桥梁隧道 / 工业厂房 / 建筑外墙
- 每个场景包含：场景痛点 + 解决方案 + 推荐产品

### 3.6 新闻资讯 / 行业动态
- 资讯列表页：展示公司动态、行业知识、政策解读
- 每条内容包含：标题 + 发布时间 + 分类标签 + 摘要 + 正文
- 可包含：防水知识科普、施工注意事项、行业标准解读等

### 3.7 联系我们
- 公司名称、地址（在地图上标注位置）
- 联系电话（可点击拨打）
- 电子邮箱
- 营业时间
- 在线询盘表单（与3.4相同，作为页面入口）
- 微信公众号二维码占位区（如有）

### 3.6 联系我们
- 公司名称、地址（在地图上标注位置）
- 联系电话（可点击拨打）
- 营业时间
- 简单的在线咨询表单（姓名 + 电话 + 留言内容）
- 电子邮箱

## 四、非功能性需求

### 4.1 响应式设计
- **PC端**：宽度 ≥ 1200px，桌面布局
- **平板端**：宽度 768px - 1199px，两栏布局
- **手机端**：宽度 < 768px，单栏布局，底部导航

### 4.2 性能要求
- 页面首屏加载时间 ≤ 3秒（图片需做懒加载优化）
- 图片格式：WebP 或 JPEG，清晰度保证

### 4.3 浏览器兼容
- Chrome / Edge（最新版）
- Safari（最新版）
- Firefox（最新版）
- 微信内置浏览器

### 4.4 域名与部署（待定）
- 域名：暂无，暂用 IP 或临时域名访问
- 部署平台：待定（可选择阿里云、腾讯云、Netlify、Vercel 等）

## 五、设计风格

### 5.1 整体风格
- **专业商务深色风格**：深色背景 + 白色文字 + 金色/橙色点缀
- **关键词**：稳重、高端、可信赖、有质感

### 5.2 配色方案（暂定）

| 颜色角色 | 色值 | 用途 |
|---------|------|------|
| 主背景色 | `#1A1A2E` | 页面背景 |
| 次背景色 | `#16213E` | 卡片背景 |
| 强调色 | `#E8A838` | 按钮、图标、高亮 |
| 文字主色 | `#FFFFFF` | 标题、重要文字 |
| 文字次色 | `#B0B0B0` | 辅助文字 |
| 边框色 | `#2D2D44` | 分割线、边框 |
| 成功色 | `#4CAF50` | 成功提示 |
| 错误色 | `#E53935` | 错误提示 |

### 5.3 字体
- 主字体：思源黑体（Noto Sans SC）或系统默认无衬线字体
- 英文：Roboto / system-ui
- 字号层级：
  - 页面标题：32-48px
  - 区块标题：24px
  - 正文：14-16px
  - 辅助文字：12px

### 5.4 图片规范
- 商品图片：比例 1:1，建议 800×800px 以上
- Banner图片：比例 16:9，建议 1920×1080px
- 图片来源：Unsplash（免费）、Picsum 或用户提供

## 六、技术方案（初步）

### 6.1 推荐方案

| 方案 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| **方案A：静态网站 + 前端框架**（推荐） | 小型官网、商品数量少 | 速度快、成本低、无需服务器 | 无动态后台 |
| 方案B：WordPress + 主题 | 快速上线 | 生态丰富、有后台 | 定制化受限 |

**推荐方案A：静态网站（HTML + CSS + JavaScript）**

| 技术选型 | 说明 |
|---------|------|
| 前端框架 | 纯 HTML + CSS + JavaScript（无框架，利于0基础学习）|
| CSS框架 | Tailwind CSS（通过CDN引入，无需构建工具）|
| 动态交互 | Alpine.js（轻量，用于轮播、购物车、表单验证）|
| 图标库 | Lucide Icons / Heroicons |
| 图片懒加载 | 原生 `loading="lazy"` |
| 部署 | Vercel / Netlify（免费，无需购买服务器）|

### 6.2 数据管理
- 商品数据：以 JSON 文件形式存储（`products.json`）
- 订单数据：本地存储（`localStorage`）+ 表单提交
- 新闻公告：以 JSON 文件形式存储（`news.json`）

### 6.3 文件结构

```
/
├── index.html          # 首页
├── about.html          # 关于我们
├── products.html       # 商品列表
├── product.html        # 商品详情（参数化）
├── cart.html           # 购物车
├── checkout.html       # 结算/下单
├── news.html           # 新闻公告列表
├── news-detail.html    # 新闻详情
├── contact.html        # 联系我们
├── css/
│   └── style.css       # 自定义样式
├── js/
│   ├── main.js         # 主逻辑（导航、购物车）
│   ├── products.js     # 商品数据加载
│   └── cart.js         # 购物车逻辑
├── data/
│   ├── products.json   # 商品数据
│   └── news.json       # 新闻数据
└── assets/
    └── images/         # 图片资源
```
